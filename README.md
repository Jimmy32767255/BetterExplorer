# BetterExplorer

一个使用Python编写的增强型文件资源管理器，具有基础桌面、文件管理和Alt+Tab切换功能，并支持多显示器。

## 功能特点

- **基础桌面**：提供桌面图标显示和右键菜单功能
- **文件管理**：支持文件浏览、复制、移动、删除等基本文件操作
- **Alt+Tab切换**：实现类似Windows的Alt+Tab窗口切换功能
- **多显示器支持**：自动识别并支持多个显示器

## 系统要求

- Windows操作系统
- Python 3.6+
- 安装依赖列表中的所有依赖

## 安装方法

1. 克隆或下载本项目到本地
2. 安装依赖包：

```bash
pip install -r requirements.txt
```

## 使用方法

运行主程序：

```bash
python main.py
```
